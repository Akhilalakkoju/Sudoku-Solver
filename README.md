# 🧩 Sudoku Solver Using Backtracking

This project is a Java-based Sudoku Solver that uses the Backtracking Algorithm to solve incomplete 9×9 Sudoku puzzles automatically.

## 🚀 Features

* Solves Sudoku puzzles automatically
* Uses recursion and backtracking
* Validates rows, columns, and 3×3 grids
* Beginner-friendly implementation

## 🛠 Technologies Used

* Java
* Backtracking Algorithm
* Recursion

## ▶️ How to Run

Compile the program:

javac SudokuSolver.java


Run the program:


java SudokuSolver


## 📚 Learning Outcomes

* Understanding recursion
* Learning backtracking
* Improving problem-solving skills
